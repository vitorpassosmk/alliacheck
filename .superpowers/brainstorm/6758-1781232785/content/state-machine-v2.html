→
CARREGANDO

→
AGUARDANDO_CTE
Aguardando emissão (externo)

→

→
FINALIZADO

Regras de transição

De Para Pré-requisito
Criação ABERTO motorista_id + veiculo_id + data_carregamento (obrigatórios)
ABERTO CARREGANDO — (nenhum, dados já estão no frete)

AGUARDANDO_CTE CTE_EMITIDO chave CT-e válida (44 dígitos, módulo 11)

O que muda vs. versão anterior

  • Estado PROGRAMADO removido do pipeline
  • Formulário de criação do frete passa a exigir motorista, veículo e data_carregamento
  • Validação que antes era na transição ABERTO→PROGRAMADO vai para o POST de criação
